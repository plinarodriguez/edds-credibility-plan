# EDDS Credibility Plan 

This directory defines the credibility plan based on the ASME V&V40 standard for an electronic drug delivery system (EDDS). 

It includes:

- Question of Interest  
- Context of Use  
- Risk Assessment  
- Credibility Goals  
- Visualizations  

This directory puts together content from a few presentations where the EDDS computatioanl model is used to illustrate credibility. 

ASME V&V 40 Standard: [Assessing Credibility of Computational Modeling through Verification and Validation: Application to Medical Devices](https://www.asme.org/codes-standards/find-codes-standards/assessing-credibility-of-computational-modeling-through-verification-and-validation-application-to-medical-devices)
