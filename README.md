# EDDS Credibility Plan 

This directory defines the credibility plan based on the ASME V&V40 standard for an electronic drug delivery system (EDDS). 

It includes:

- Question of Interest  
- Context of Use  
- Risk Assessment  
- Credibility Goals  
- Visualizations of Concepts.  

This directory puts together content from a few presentations where the EDDS computatioanl model is used to illustrate credibility.
